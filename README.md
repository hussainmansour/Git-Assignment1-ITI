# Git-Assignment1-ITI

Welcome! This repository is a hands-on exercise created at ITI to help us learn Git fundamentals by actually using them. We practice cloning, branching, committing, merging, and resolving conflicts while keeping the project simple and focused on version control skills. (Wink wink )

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/hussainmansour/Git-Assignment1-ITI.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Git-Assignment1-ITI
   ```

3. Open the `index.html` file in your web browser:
   - You can simply double-click the `index.html` file, or
   - Open it from the command line: `open index.html` (macOS) or `start index.html` (Windows) or `xdg-open index.html` (Linux)
